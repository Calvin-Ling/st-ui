# Z-Index Tokens

Z-indexes are used to stack components in a logical manner.

| ----------------------- | ----- |
| `--st-z-index-drawer`   | 700   |
| `--st-z-index-dialog`   | 800   |
| `--st-z-index-dropdown` | 900   |
| `--st-z-index-toast`    | 950   |
| `--st-z-index-tooltip`  | 1000  |
